# Spectral Clustering Code for Cheruvelil et al. 

## Data 

Get data from the LTER data portal and unzip to the same directory as the code. It should include the following files:

* terrData.csv
* freshData.csv
* NB_18876.csv
* IslandIdx.csv
* latLong18876.csv
